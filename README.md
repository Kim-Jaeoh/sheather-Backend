## 📋 프로젝트

>

- 해당 프로젝트는 기온별 옷차림을 공유하고 날씨를 확인할 수 있는 웹앱입니다.
  날씨(기온)가 애매하여 어떤 옷을 입고 나가야 할지 고민이 되고, 사람들은 어떻게 입었을까 라는 생각이 들어 '기온별 옷차림'이라는 주제를 정해 제작하게 되었습니다.
  - 기존 피드에 업로드 되어 있는 이미지 출처 - 무신사 브랜드 스냅
  
<br>
